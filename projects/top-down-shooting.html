🎮 프로젝트 개요

탕탕특공대를 모작으로 한 탑다운 슈팅게임입니다. 5스테이지를 완성하고 ObjectPool · UI 시스템 · 전투 로직을 직접 구현했습니다.
PLATFORM
PC (Unity)

5스테이지

🛠️ 구현 내용

탑다운 슈팅 게임 1인 개발 (5스테이지 완성)
ObjectPool 시스템 구현으로 런타임 메모리 최적화
UI 시스템 개발 (체력바, 스테이지 전환, 게임오버 화면)
인게임 전투 로직 구현 (플레이어 조작, 총알 발사, 적 패턴)

⚙️ 핵심 시스템

01ObjectPool 기반 총알·적 재사용

총알과 적 오브젝트를 매 프레임 생성/파괴 대신 ObjectPool로 재사용해 GC 부하를 제거했습니다.

02플레이어 전투 시스템

WASD 이동, 마우스 방향 조준, 총알 발사 로직을 구현했습니다. 무기별 발사 속도와 데미지를 데이터로 분리했습니다.

03적 AI 패턴

적의 이동 패턴과 공격 로직을 구현했습니다. 스테이지 진행에 따라 적의 수와 속도가 증가합니다.

04UI 시스템

체력바, 스테이지 전환 연출, 게임오버 화면을 구현했습니다.
